# 로또 번호 추첨기 웹 서비스

## 개요

이 프로젝트는 사용자가 자신의 로또 번호를 입력하고, 여러 게임의 추첨 결과를 시뮬레이션해볼 수 있는 간단한 웹 애플리케이션입니다. 현대적인 웹 표준 기술(HTML, CSS, JavaScript)을 사용하여 제작되었으며, 사용자 친화적인 인터페이스와 시각적으로 매력적인 디자인을 제공하는 것을 목표로 합니다.

---

## 디자인 및 기능

### 1. **시각적 디자인**

-   **레이아웃**: 중앙 정렬된 깔끔한 단일 컬럼 레이아웃을 사용합니다.
-   **색상 팔레트**: 메인 색상으로 신뢰감을 주는 파란색(`#4a90e2`)과 활기찬 느낌의 민트색(`#50e3c2`)을 사용하여 시각적 즐거움을 더했습니다.
-   **그림자 효과**: 카드와 버튼에 부드러운 그림자(`box-shadow`)를 적용하여 입체감을 주고, 각 요소를 시각적으로 구분합니다.
-   **폰트**: 시스템 기본 폰트를 사용하여 어떤 기기에서든 가독성이 좋도록 했습니다.
-   **상호작용**: 사용자가 입력 필드에 포커스하거나 버튼에 마우스를 올렸을 때 색상과 그림자가 부드럽게 변하는 효과를 추가하여 상호작용 경험을 향상시켰습니다.

### 2. **핵심 기능**

-   **사용자 번호 입력**: 사용자는 1부터 45까지의 숫자 6개를 중복 없이 입력할 수 있습니다.
    -   유효성 검사: 숫자가 아니거나 범위를 벗어난 값을 입력하면 경고 메시지를 표시합니다.
-   **로또 번호 추첨**: '추첨하기' 버튼을 누르면 한 번에 5개의 게임이 실행됩니다.
    -   각 게임마다 1부터 45 사이의 무작위 숫자 6개가 추첨됩니다.
-   **결과 표시**: 각 게임별로 다음 정보가 표시됩니다.
    -   사용자가 입력한 번호
    -   추첨된 번호
    -   일치하는 번호의 개수
    -   일치하는 번호는 눈에 띄는 색상(금색)으로 강조 표시됩니다.

### 3. **구현 기술**

-   **HTML**: 웹 페이지의 구조를 정의합니다. (`index.html`)
-   **CSS**: 시각적 스타일과 레이아웃을 담당합니다. (`style.css`)
-   **JavaScript**: 사용자 상호작용과 로또 추첨 로직을 구현합니다. (`main.js`)

---

## 현재 요청 계획

1.  **`main.js` 작성**: 로또 번호 생성, 사용자 번호와 비교, 결과 표시 등 핵심 로직을 구현했습니다.
2.  **`index.html` 작성**: 사용자가 번호를 입력하고 결과를 확인할 수 있는 페이지의 구조를 정의했습니다.
3.  **`style.css` 작성**: 현대적이고 시각적으로 매력적인 디자인을 적용했습니다.
4.  **`blueprint.md` 작성**: 프로젝트의 목적, 디자인, 기능, 구현 계획을 문서화했습니다.
